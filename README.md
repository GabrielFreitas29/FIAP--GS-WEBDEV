# SentinelaOrbital

## Sobre o Projeto

O SentinelaOrbital é uma plataforma desenvolvida para auxiliar no monitoramento de desastres climáticos, utilizando dados de satélites, sensores IoT e APIs climáticas para gerar alertas preventivos.

## Funcionalidades

- Landing Page informativa
- Slideshow com imagens do tema
- Troca de tema (Verde, Azul e Escuro)
- Formulário com validação
- Quiz educativo com 10 perguntas
- Resultado final do quiz

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
